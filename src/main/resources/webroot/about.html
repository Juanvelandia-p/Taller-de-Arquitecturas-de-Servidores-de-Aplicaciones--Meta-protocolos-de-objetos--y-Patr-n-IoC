Acerca de MicroSpringBoot

¿Qué es MicroSpringBoot?

MicroSpringBoot es un framework web ligero escrito en Java que implementa los conceptos básicos de Inversión de Control (IoC) y arquitectura REST, inspirado en Spring Boot.

Características principales:

  • Servidor HTTP: Implementa un servidor web básico usando ServerSocket de Java
  • Anotaciones personalizadas: @RestController, @GetMapping, @RequestParam
  • Reflexión: Uso de Java Reflection para escanear y cargar controladores dinámicamente
  • Enrutamiento: Sistema de rutas que mapea URLs a métodos de controlador
  • Archivos estáticos: Capacidad de servir HTML, CSS, JS, PNG, JPG, etc.
  • Query Parameters: Soporte para parámetros de consulta con valores por defecto

Tecnologías utilizadas:

  • Java 17
  • Maven
  • Java Reflection API
  • Java Networking (ServerSocket, Socket)
