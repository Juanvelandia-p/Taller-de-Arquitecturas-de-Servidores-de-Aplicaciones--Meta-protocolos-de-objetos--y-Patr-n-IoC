Bienvenido a MicroSpringBoot

Este es un servidor web simple implementado en Java con soporte para IoC y arquitectura REST.

Características:

  • Servidor HTTP básico
  • Soporte para anotaciones @RestController y @GetMapping
  • Soporte para @RequestParam con valores por defecto
  • Servir archivos estáticos (HTML, PNG, etc.)
  • Escaneo automático de controladores en el classpath
